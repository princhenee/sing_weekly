Taking the Blame Game out of Data Centers Operations with NetPoirot

Summary:

In this paper, the authors pose the question: how much can you infer about
a failure in the data center using TCP statistics collected at one of the
endpoints. Using an agent that captures TCP statistics NetPoirot devises a
classification algorithm that identifies the root cause of failure using this
information at a single endpoint. Based on the inferred results by applying
the model to TCP metrics, data center network operators may be able to predict
potential failures.
